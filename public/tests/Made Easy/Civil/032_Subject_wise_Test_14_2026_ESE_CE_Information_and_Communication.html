Consider the following statements:
1. USSD stands for Unstructured Short Service Data.
2. SMS is a store and forward technology, whereas USSD texts and interactions are not stored on the mobile phone.
3. USSD messages are up to 282 alphanumeric characters long.
Which of the above statements is/are NOT correct?

Correct answer: D
Solution

(d)
• USSD stands for Unstructured Supplementary Service Data is a communication protocol used in GSM networks for sending short text messages.
• USSD messages are up to 182 alphanumeric characters long.
• SMS works on a store-and-forward basis; i.e. when one device sends an SMS to another, no dedicated communication channel is opened, and there is always a short delay in the recipient getting the message. By contrast, when a device sends out a USSD request, a transparent
channel is automatically established between sender and recipient. In this way, the recipient can respond to the sender’s requests in real time. Thus, USSD texts and interactions are not stored on the mobile phone.

Codes:
(a) Both Statement (I) and Statement (II) are individually true and Statement (II) is the correct
explanation of Statement (I).
(b) Both Statement (I) and Statement (II) are individually true but Statement (II) is NOT the
correct explanation of Statement (I).
(c) Statement (I) is true but Statement (II) is false.
(d) Statement (I) is false but Statement (II) is true.

Statement (I): Medium Access Control is a sublayer in the Physical layer which deals with the issue of access control to the shared channel.
Statement (II): The network layer is necessary if two systems are connected to different links with connecting devices between the links to ensure the source-to-destination delivery.

Correct answer: D
Solution

(d)
• Medium Access Control (MAC) is a sublayer in the Data Link layer which deals with the issue
of access control to the shared channel.
• The network layer ensures that each packet gets from its point of origin to its final destination.

However, if the two systems are attached to different networks (links) with connecting devices
between the networks (links), there is a need for the network layer to accomplish source to destination delivery.
