In the final simplex tableau of linear programming problem (LPP), it is found that zj - cj = 0 for at least one non-basic variable. Which of the following conclusions is correct?

Correct answer: C
Solution

(c)

In reference to Kendall’s notations (a | b | c = d | e | f ) used for queuing theory, which of the following statement is correct?

Correct answer: C
Solution

(c)
In Kendall's notation,
'a' represents arrival distribution.
'b' represents diparture (or service time) distribution.
'c' represents number of servers in the system.
'd' represents service discipline.
'e' represents maximum number of customers allowed in the system.
'f' represents calling source or population.

A basic feasible solution to a m × n transportation problem is said to be non-degenerate if

Correct answer: A
Solution

(b, c)
A basic feasible solution to a m × n transportation problem is said to be non-degenerate if
(a) The total number of non-negative allocations is exactly m + n – 1.
(b) There m + n – 1 allocations are in independent position.
