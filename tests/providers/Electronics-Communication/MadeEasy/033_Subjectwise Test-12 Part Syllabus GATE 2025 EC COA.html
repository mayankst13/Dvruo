Consider the following statements:
(i) In case of pipeline processor, loop buffer is very high speed memory maintained by instruction fetch.
(ii) The process of fetching and executing instructions one at a time, in the order of increasing addresses is known as straight line sequencing.
(iii) In RISC instruction set, all arithmetic/logic instructions must be register based.
Which of the above statements are correct?

Correct answer: C
Solution

(c)
• A loop buffer is a small, very-high-speed memory maintained by the instruction fetch stage of the pipeline and containing the ‘n’ most recently fetched instructions, in sequence. The instructions fetched in sequence will be available without the usual memory access time well suited to dealing with loops.
• In straight line sequencing, the instruction of a program is executed in a sequential manner (i.e. every time PC is incremented by a fixed offset).
• RISC architectures rely on registers for operand storage and manipulation, reducing memory access overhead. Therefore, the arithmetic/logic instructions are register based.
Hence, all the above statements are correct.

Q18 Not checked
Technical • ELECTRONICS ENGINEERING • +2 / -0.66
Not attempted

The following diagram shows, which addressing mode for 6802 MPU?

Correct answer: C
Solution

(c)
For 6802 MPU, in direct addressing, the address of the operand is contained in the second byte of the instruction. Direct addressing allows the user to directly address the lowest 256 bytes of the memory, i.e, locations 0 through 255. Enhanced execution times are achieved by storing data in these locations. In extended addressing mode, the address contained in the second byte of the instruction is used as the higher eight bits of the address of the operand. The third byte of the instruction is used as the lower eight bits of the address for the operand. Thus, the given diagram represents extended addressing mode.
